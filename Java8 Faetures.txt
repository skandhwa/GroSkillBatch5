Stream API 
-------------------------
Stream API is used to process collections of data in an functional and declarative way 
java.utl.stream package

filtering, mapping,sorting,reducing
To process group of object then we are going to use stream API
Does not store any data only be used to perform operations on data

Collect and Collectors
